Intro

Installable via npm install --save peer-info, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save peer-info

You can then require() peer-info as normal:

const peerInfo = require('peer-info')

In the Browser

peer-info should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of peer-info, including index.js and index.min.js, are included in the /dist folder. peer-info can also be found on unpkg.com under

  • https://unpkg.com/peer-info/dist/index.min.js
  • https://unpkg.com/peer-info/dist/index.js
